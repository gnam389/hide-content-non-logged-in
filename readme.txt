== Description ==
The plugin can hide any content from non-logged in users. The code is simple, compact, does not take up storage space or system resources. Works with shortcodes like CF7.

== Installation ==
1. Upload `hide-content-non-logged-in` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use shortcode `[hide_content]...[/hide_content]` to hide content for non-logged in users.
